# ae-test-task

**Requirements**

Python 3.7.1

**Install dependencies**

```
pip install -r requirements.txt
```

**RUN Application**

```
python main.py <original_file_path.html> <other_sample_file_path.html>
```

**For example**

```
python main.py res/sample-0-origin.html res/sample-1-evil-gemini.html
```
